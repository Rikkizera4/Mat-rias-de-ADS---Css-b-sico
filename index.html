ADS - 2024

O curso de Análise e Desenvolvimento de Sistemas (ADS) é uma formação superior que visa capacitar os alunos para atuar na criação, desenvolvimento e manutenção de sistemas computacionais. Durante o curso, os estudantes aprendem a projetar e implementar soluções tecnológicas que atendem às necessidades de empresas e organizações.

Disciplinas

  • Desenvolvimento de aplicações para internet

    Estudo de linguagem de programação para desenvolvimento de aplicações voltados para Internet. Desenvolvimento de aplicações Web cliente/servidor. Tecnologias Web. HTML. Linguagens de programação para Web. Arquitetura de aplicações Web. Usabilidade, tendências e frameworks para desenvolvimento Web.

    Prof: Deivid Sala: 112

    Análise e Modelagem de Sistemas

    Processo de estudar requisitos de sistemas de software e criar modelos representativos (casos de uso, diagramas UML) para estruturar e compreender a arquitetura e o comportamento do sistema antes do desenvolvimento.

    Prof: Rafael Fonseca Queiroz Sala: 112

    Atividades Práticas Interdisciplinares de Extensão I

    Aplicação prática de conhecimento teórico em projetos que integram diferentes disciplinas, focando no desenvolvimento de habilidades de resolução de problemas, inovação e colaboração em contextos reais.

    Prof: Anderson Martins da Silva Sala: 112

    Banco de Dados

    Sistema organizado de armazenamento de dados que permite acesso, gerenciamento e atualização eficientes. Inclui bancos de dados relacionais (SQL) e não relacionais (NoSQL).

    Prof: Rafael Fonseca Queiroz Sala: 112

    Front-End Frameworks

    Ferramentas ou bibliotecas que facilitam o desenvolvimento de interfaces de usuário para aplicativos web, fornecendo componentes pré-construídos e padrões de design. Exemplos incluem React, Angular e Vue.js.

    Prof: Deivid Francis de Oliveira Sala: 112

    Programação Orientada a Objetos e Estrutura de Dados

    O desenvolvimento de software que organiza o código em unidades chamadas objetos, que representam entidades do mundo real ou abstratas, encapsulando dados e métodos que operam sobre esses dados. Esse paradigma facilita a reutilização de código, modularidade e manutenção, promovendo a criação de sistemas robustos e escaláveis

    Prof: Rafael Fonseca Queiroz Sala: 112

    Transformação Digital, Sistemas Computacionais e o Futuro da Tecnologia

    Estudo de linguagem de programação para desenvolvimento de aplicações voltados para Internet. Desenvolvimento de aplicações Web cliente/servidor. Tecnologias Web. HTML. Linguagens de programação para Web. Arquitetura de aplicações Web. Usabilidade, tendências e frameworks para desenvolvimento Web.
